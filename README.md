# Stackr-mvp

A quick and dirty MVP for Stackr.

## Running the Application

To run the Rollapp, use the following commands:
    
    ```bash
    cd stackr-mvp
    go get .
    go run .
    ```

To send a signed msg to the rollapp, use the following command:

    ```bash
    cd stackr-mvp/scripts
    go run sendTx.go
    ```
